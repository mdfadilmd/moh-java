1. Muat turun (download) JDK dari laman web Oracle

2. Pasang (install)
	- pasang ke C:\program files\java\jdk..

3. Set Environment variable path point ke C:\program files\java\jdk..\bin

4. compile --> javac <nama fail>.java di GIT Bash

5. run --> java <nama fail> di GIT Bash
